# 별보기 지수 (Stargazing Index)

날씨 데이터로 오늘 밤 별 관측 조건을 0~100%로 보여주는 웹 앱.
애플 날씨 앱 느낌의 감성적인 UI에, 날씨에 따라 살아 움직이는 하늘 배경(별·구름·실제 달 위상)을 입혔습니다.

## 실행

```bash
npm install
npm run dev      # 개발 서버 (기본 http://localhost:5173)
npm run build    # 프로덕션 빌드 → dist/
npm run preview  # 빌드 결과 미리보기
```

## 지수 계산식

```
I = max(0, B − Δhum − Δpm − Δwind − Δmoon)
```

- **B (기본 점수, 운량)**: `100 × (1 − 운량/100)^1.5` — 구름이 점수의 상한을 결정
- **Δhum (습도)**: 40% 이하 0, 100%에서 최대 −20
- **Δpm (초미세먼지)**: 75㎍/㎥ 이상 최대 −10
- **Δwind (풍속)**: 3m/s 이하 0, 15m/s 이상 최대 −7
- **Δmoon (달)**: 밝은 면 비율 × 밤 동안 고도 노출도, 최대 −8

"오늘 밤"과 앞으로 3일은 각 밤의 **저녁 19시 ~ 다음날 05시** 구간 평균으로 계산합니다.
(`src/lib/score.ts` 에서 계수 조정 가능)

## 데이터 출처

- **Open-Meteo** (API 키 불필요): 운량·습도·풍속·기압·기온, 대기질(PM2.5), 위치 검색(지오코딩)
- **SunCalc**: 달 위상·밝은 면 비율·고도

## 폰트

Pretendard를 jsDelivr CDN으로 불러옵니다 (`src/styles.css` 최상단 `@import`).
self-host 하려면 그 한 줄을 로컬 폰트 경로로 교체하세요.

## 위치 저장

검색한 위치는 `localStorage`(키: `stargazing.location.v1`)에 저장되어 다음 방문 때 복원됩니다.
